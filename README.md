# 割り勘LINE Bot (Warikan LINE Bot)

旅行用の割り勘管理 LINE Bot。Google Apps Script (GAS) + Google スプレッドシートで動作。
**「グループに招待するだけで使える」**ことを目指した設計。

## ✨ できること

* 招待した瞬間に Bot が使い方を自動で説明
* `@`プレフィックスでサクッと支払い記録
* 4行目以降に対象者を書けば限定割り勘
* 「精算」と打つだけで履歴のメンバーで自動精算
* Flex Message で見やすい精算結果・履歴カード
* タイポっぽい名前を検知して警告
* 過去の精算記録を見返す隠しコマンド

## 🔧 使い方

### 招待後

Bot がグループに参加すると、ウェルカムメッセージ + メニューボタンが表示されます。
Bot をメンションするか `ヘルプ` と送れば、いつでもメニューを呼び出せます。

### 主なコマンド

#### 支払いを記録（@）

```
@（支払った人）
（金額）
（内容）  ※省略可
```

例:
```
@たろう
3000
ランチ
```

**対象者を絞りたいときは 4 行目以降に列挙**:

```
@たろう
6000
カラオケ
たろう
はなこ
```

→ たろうとはなこの 2 人だけで割り勘。

#### 精算

「精算」とだけ送ると、履歴のメンバーで自動精算するか確認されます。
0円参加者を含めたいときは:

```
精算
（参加者A）
（参加者B）
...
```

#### その他

| コマンド | 用途 |
|---|---|
| `履歴` | 未精算の記録一覧（最新10件、Flex Message） |
| `メンバー` | 未精算で記録されている人の一覧 |
| `取消` | 最近の記録から選んで取消（誰の記録でもOK） |
| `ヘルプ` / `使い方` | メニューを表示 |
| `過去の履歴` | 隠しコマンド: 精算済の過去記録を表示 |

## 🚀 セットアップ

1. LINE Developers で Messaging API のチャネルを作成
2. Google スプレッドシートを新規作成
3. GAS プロジェクトを新規作成し、`clasp push` でこのリポジトリのコードを送信
4. GAS の「スクリプトプロパティ」に以下を設定:
   * `CHANNEL_ACCESS_TOKEN`: LINE Developers から取得したトークン
   * `SPREADSHEET_ID`: スプレッドシートのURLから取得したID
   * `SHEET_NAME`: シート名（省略時 `シート1`）
5. GASプロジェクトを「ウェブアプリ」として「デプロイ」（誰でもアクセス可能、自分として実行）
6. デプロイURLを LINE Developers の Webhook URL に設定
7. LINE 側で「Webhook の利用」を ON、「応答メッセージ」を OFF
8. グループ・複数人トークへの参加を許可する設定を ON
